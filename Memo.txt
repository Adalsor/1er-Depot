                                                      Memo Git

Configurer Git:

git config --global user.name "Your Name"                            Config ton Nom
git config --global user.email "you@example.com"                     Config ton Mail
git config --list                                                    Voir tes config

Dossier Git:

mkdir myproject                                                      Créer un dossier 
cd myproject                                                         Se déplacer dans les dossiers
ls                                                                   Voir les dossier/fichier là où on se trouve

Tag Git:

git tag <tagname>                                                    Créer une balise légère
git tag -a <tagname> -m "message"                                    Créer une balise annotée
git tag <tagname> <commit-hash>                                      Marquer un commit spécifique
git tag                                                              Liste des balises
git show <tagname>                                                   Afficher les détails de la balise
git push origin v1.0                                                 Pousser une seul balise vers le référentiel
git push --tags                                                      Pousser toutes les balises
git tag -d v1.0                                                      Supprimer une balise localement
git push origin --delete tag v1.0                                    Supprimer une balise du référentiel distant

Commande Git:

git init                                                             Initialiser Git
git status                                                           Verifier quel fichiers sont suivis
git add <file>                                                       Inclus le fichier pour le prochain commit
git add --all                                                        Inclus tout les fichiers
git restore --staged <file>                                          Annule le add
git commit -m "message"                                              Point de sauvegarde de vos projets
git commit -a -m "message"                                           Tout sauvegarder (ignorer add)
git log                                                              Voir l'historique des commits
git log --oneline                                                    Voir l'historique en une ligne
git push origin                                                      Partager les modifications
git push --tags                                                      Envoie toutes les balises locales vers GitHub
git push origin v1.0                                                 Pour pousser une balise spécifique
git commit --amend -m "Corrected message"                            Changer le nom du dernier commit
git diff                                                             Voir les différences non validées
git clone <url du dépôt> [nom du dossier]	                     Cloner un dépôt Git existant
git fetch origin <nom de la branch>                                  Récupère les fichiers distant
git pull origin <nom de la branch>                                   Effectue un git fetch + un git merge

git show <commit>                                                    Afficher les détails d'un commit spécifique
git stash                                                            Commit temporaire
git help <command>                                                   Voir l'aide pour une  commande
git help --all                                                       Lister toutes les commandes Git possibles
git restore <file>                                                   Annulez les modifications avant le add
git restore --staged <file>                                          Annuler la mise en scène d'un fichier

Branch Git:

git branch "nom de la branche"                                       Créé une branche
git checkout "nom de la branche"                                     Naviguer dans la branche
git checkout master                                                  Aller à la branche master
git branch -d "nom de la branche"                                    Supprimer la branche
git branch                                                           Lister les branches existantes
git merge <nom de la branch>                                         Fusionner une branche dans branche actuelle
git merge --abort                                                    Annuler une fusion en cours

SSH:

eval $(ssh-agent -s)                                                 Si on a jamais utilisé de clé sur notre ordi
ssh-keygen -t rsa -b 4096 -C "your@email.com"                        Générer clé SSH (dans id_rsa)
ssh-add ~/.ssh/id_rsa                                                Ajouter la clé à votre Git
clip < ~/.ssh/id_rsa.pub                                             Copier la clé publique (dans id_rsa.pub)
ssh-add -l                                                           Lister les clés charghé
ssh-add -d ~/.ssh/id_rsa                                             Supprime la clé
ssh -T git@github.com                                                Verifier votre connexion SSH à GitHub 
git remote add origin git@github.com:your-username/your-repo.git     Ajouter l'origine distante
git remote set-url origin git@github.com:your-username/your-repo.git Mettre à jour
git push -u origin main                                              push en passant sa branch en upstream
git branch -M "nom de la branche"                                    Renommer une branche
